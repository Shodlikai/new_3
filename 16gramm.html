Xayoliy vaziyatlar, afsuslanishlar va imkonsiz shartlar uchun qo'llanma

0%

Shart Ergash Gaplar Qoidalari (Type 2 & 3)

Conditional Type 2 (Second Conditional)

Ishlatilishi: Hozirgi yoki kelajakdagi **xayoliy, noreal, ehtimoli juda past** vaziyatlar va ularning xayoliy natijalarini tasvirlash uchun ishlatiladi. Ko'pincha maslahat berish ("If I were you..." - "O'rningda bo'lganimda...") yoki orzu qilish uchun qo'llaniladi.

Formula: If + Past Simple, ... would + V1

If I had the time, I would go to the movie with you. (Agar vaqtim bo'lganda edi, sen bilan kinoga borardim - lekin hozir vaqtim yo'q).

If I were you, I would take the yellow dress. (Agar o'rningda bo'lganimda, sariq ko'ylakni olardim - maslahat).

Conditional Type 3 (Third Conditional)

Ishlatilishi: O'tgan zamonda sodir bo'lmagan, **imkonsiz** vaziyatlar va ularning amalga oshmagan natijalarini tasvirlash uchun ishlatiladi. Ko'pincha o'tmishdagi harakatlar uchun **afsuslanish, tanqid qilish yoki boshqacha natijani tasavvur qilish** uchun qo'llaniladi.

Formula: If + Past Perfect, ... would + have + V3

If I had left earlier, I would have caught the train. (Agar ertaroq jo'nab ketganimda edi, poyezdga ulgurgan bo'lardim - lekin kech ketdim va ulgura olmadim).

If we hadn't lost our way, we would have arrived sooner. (Agar yo'ldan adashmaganimizda edi, ertaroq yetib kelgan bo'lardik).

Shart Gap Turlari O'rtasidagi Farqlar

1. Real vs. Noreal Shartlar (Type 1 vs. Type 2)

Bu ikki tur shart gapning asosiy farqi ularning qanchalik real ekanligidadir.

  • Type 1 (Real Ehtimol): Kelajakda sodir bo'lishi mumkin bo'lgan vaziyatlar uchun ishlatiladi. Shart bajarilishiga ishonch bor.
  • Type 2 (Noreal Xayolot): Hozir yoki kelajakda sodir bo'lishi deyarli imkonsiz bo'lgan xayoliy vaziyatlar uchun ishlatiladi.

Type 1: If I win the lottery, I will buy a new car. (Agar lotereya yutsam, yangi mashina sotib olaman - Bu real ehtimol).

Type 2: If I won the lottery, I would buy a new car. (Agar lotereya yutganimda edi, yangi mashina sotib olardim - Bu shunchaki orzu, yutishimga ishonmayman).

2. Hozirgi/Kelajakdagi Xayolot vs. O'tmishdagi Afsus (Type 2 vs. Type 3)

Bu ikki noreal shart gapning asosiy farqi ular ishora qilayotgan vaqtdadir.

  • Type 2 (Hozirgi/Kelajak): "Agar hozir shunday bo'lganda edi..." degan ma'noni beradi. Hozirgi vaqtdagi haqiqatga zid bo'lgan xayolot.
  • Type 3 (O'tmish): "Agar o'tmishda shunday bo'lganida edi..." degan ma'noni beradi. O'zgartirib bo'lmaydigan o'tmish haqidagi afsus.

Type 2: If I spoke French, I would apply for that job. (Agar fransuzcha gapira olganimda edi, o'sha ishga ariza topshirardim - lekin hozir gapira olmayman).

Type 3: If I had spoken French, I would have applied for that job. (Agar fransuzcha gapira olganimda edi, o'sha ishga ariza topshirgan bo'lardim - o'tmishda shunday imkoniyat bo'lgan, lekin til bilmaganim uchun topshirmaganman).

Shart Ergash Gap Tuzilmalari (Darak, Inkor, So'roq)

Conditional Type 2

If + S + Past Simple, S + would + V1.
If + S + didn't + V1, S + would + V1.
Would + S + V1 if + S + Past Simple?

(+) If I had the time, I would go to the movie.

Agar vaqtim bo'lganda, kinoga borardim.

(-) If he didn't speak so fast, you could understand him.

Agar u bunchalik tez gapirmaganda, siz uni tushuna olardingiz.

Conditional Type 3

If + S + had + V3, S + would have + V3.
If + S + hadn't + V3, S + would have + V3.
Would + S + have + V3 if + S + had + V3?

(+) If I had been free yesterday, I would have joined you.

Agar kecha bo'sh bo'lganimda, senga qo'shilgan bo'lardim.

(-) If we hadn't lost our way, we would have arrived sooner.

Agar biz yo'limizdan adashmaganimizda, ertaroq kelgan bo'lardik.

Agar biz yo'limizdan adashmaganimizda, ertaroq kelgan bo'lardik.

Agar soat oltida qo'ng'iroq qilganingizda edi, siz uni o'sha yerdan topishingiz mumkin edi.

Muhim Qoidalar va Xatolar (Conditionals 2 & 3)

🚫 XATO 1: `if` ergash gapida `would` ishlatish
If I would have more time, I would travel.
If I had more time, I would travel.

Eng asosiy qoida: `if` bilan boshlangan shart qismida hech qachon `would` yoki `would have` ishlatilmaydi.

🚫 XATO 2: Type 2 va Type 3 ni aralashtirish
If she had studied, she would pass.
If she had studied, she would have passed.

Qoida: Agar `if` qismida Past Perfect (had + V3) bo'lsa, bosh gapda ham unga mos ravishda `would have + V3` bo'lishi kerak.

🚫 XATO 3: Type 2 da 'was' o'rniga 'were'
If I was you, I would apologize.
If I were you, I would apologize.

Qoida: Type 2 da maslahat berishda yoki xayoliy vaziyatlarda barcha shaxslar uchun `to be` fe'lining `were` shaklini ishlatish grammatik jihatdan to'g'riroq hisoblanadi.

🚫 XATO 4: Type 3 da `have` ni unutish
If I had known, I would gone.
If I had known, I would have gone.

Qoida: Type 3 ning bosh gapida `would` va `V3` o'rtasida `have` yordamchi fe'li bo'lishi shart.

🚫 XATO 5: Type 3 da Past Perfect o'rniga Past Simple
If I knew about it yesterday, I would have helped.
If I had known about it yesterday, I would have helped.

Qoida: Agar bosh gap `would have + V3` bilan kelayotgan bo'lsa, `if` qismi Past Perfect (`had + V3`) bo'lishi kerak.

🚫 XATO 6: Og'zaki nutqdagi xato ('would of')
We would of gone if you had called.
We would have gone if you had called.

Qoida: Og'zaki nutqda `would've` (would have) ning qisqartmasi `would of` kabi eshitilishi mumkin, lekin yozuvda bu jiddiy xato. Har doim `would have` deb yoziladi.

🚫 XATO 7: Type 1 va Type 2 ni adashtirish
If I have a million dollars, I would quit my job.
If I had a million dollars, I would quit my job.

Qoida: Bosh gapda 'would' kelishi bilan bu xayoliy vaziyat (Type 2) ekanligi ma'lum bo'ladi. Shuning uchun 'if' qismida Present Simple emas, Past Simple ishlatilishi kerak.

🚫 XATO 8: Type 3 da V3 ni unutish
If he had left earlier, he would have catch the train.
If he had left earlier, he would have caught the train.

Qoida: Type 3 ning bosh gapida 'would have' dan keyin fe'lning 3-shakli (Past Participle - V3) kelishi shart.

🚫 XATO 9: `unless` bilan noreal gaplar yasash
I wouldn't have passed the exam unless I hadn't studied.
I wouldn't have passed the exam if I hadn't studied.

Qoida: 'Unless' (if... not) odatda faqat real shartlar (Type 1) uchun ishlatiladi. Noreal (Type 2, 3) shartlar uchun odatda 'if... not' ishlatilgani ma'qul.

🚫 XATO 10: Ikkala qismda `would` ishlatish
If she would have invited me, I would have come.
If she had invited me, I would have come.

Qoida: Bu keng tarqalgan xato. Yodda tuting: 'would' faqat natijani bildiruvchi bosh gapda keladi, 'if' bilan boshlangan shart qismida emas.
