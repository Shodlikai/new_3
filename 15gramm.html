Haqiqatlar, odatlar va kelajakdagi real ehtimollar uchun shart gaplar

Shart Ergash Gaplar Qoidalari (Type 0 & 1)

Conditional Type 0 (Zero Conditional)

Ishlatilishi: 100% aniq bo'lgan, doimiy takrorlanadigan, o'zgarmas qonuniyatlar, ilmiy faktlar va umumiy haqiqatlar uchun ishlatiladi. "Agar shunday bo'lsa, doim bunday bo'ladi" degan ma'noni beradi.

Formula: If + Present Simple, ... Present Simple

If you heat water to 100°, it boils. (Agar suvni 100° gacha qizdirsangiz, u qaynaydi - bu ilmiy fakt).

If it rains, the streets get wet. (Agar yomg'ir yog'sa, ko'chalar ho'l bo'ladi - bu doimiy qonuniyat).

Conditional Type 1 (First Conditional)

Ishlatilishi: Kelajakda sodir bo'lishi mumkin bo'lgan, real va ehtimoli yuqori bo'lgan voqea-hodisalar uchun ishlatiladi. "Agar shunday bo'lsa, shunday bo'ladi/bo'lishi mumkin" degan ma'noni beradi.

Asosiy Formula: If + Present Simple, ... will + V1

If I finish work early, I will come home. (Agar ishni erta tugatsam, uyga kelaman - bu real ehtimol).

If he runs, he'll get there in time. (Agar u yugursa, u yerga vaqtida yetib boradi).

Type 1 ning boshqa ko'rinishlari

Bosh gapda (natija qismida) will o'rniga boshqa strukturalar ham kelishi mumkin:

  • Modal Fe'llar (may, might, can, must, should): Ehtimollik, imkoniyat, majburiyatni bildirish uchun.
    If it stops snowing, we can go out. (Agar qor yog'ishi to'xtasa, biz tashqariga chiqa olamiz).
  • Buyruq mayli (Imperative): Maslahat yoki buyruq berish uchun.
    If you see Tom tomorrow, tell him to ring me. (Agar ertaga Tomni ko'rsang, unga qo'ng'iroq qilishimni ayt).
  • Present Continuous/Perfect: Shart qismida hozir davom etayotgan yoki yaqinda tugagan harakatni bildirish uchun.
    If you are looking for Peter, you'll find him upstairs. (Agar siz Peterni qidirayotgan bo'lsangiz, uni yuqoridan topasiz).

Type 0 va Type 1 o'rtasidagi farqlar

Asosiy farq: Aniq Fakt vs. Real Ehtimol

Bu ikki tur shart gapning asosiy farqi ularning ifodalaydigan ma'no darajasida yotadi: biri 100% aniq, ikkinchisi esa ehtimoliy.

Conditional Type 0 (Zero Conditional)

Bu turdagi shart gaplar **doimiy haqiqat** haqida gapiradi. Agar shart bajarilsa, natija har doim bir xil va muqarrar bo'ladi. Bu umumiy qoida, odat yoki ilmiy qonuniyatdir.

If you don't water plants, they die.

Agar o'simliklarni sug'ormasangiz, ular o'ladi. (Bu biologik qonun, har doim shunday bo'ladi).

Conditional Type 1 (First Conditional)

Bu turdagi shart gaplar esa **kelajakdagi real ehtimol** haqida gapiradi. Agar shart bajarilsa, natijaning sodir bo'lish ehtimoli yuqori, lekin 100% kafolatlanmagan. Bu ma'lum bir vaziyatga xos bashorat, va'da yoki ogohlantirishdir.

If you don't water these plants, they will die.

Agar mana bu o'simliklarni sug'ormasangiz, ular o'ladi. (Bu umumiy qonun emas, aynan mana shu vaziyatga xos ogohlantirish).

Shart Ergash Gap Tuzilmalari

Conditional Type 0

If + S + Present Simple, S + Present Simple.

If you heat ice, it turns to water.

Agar muzni isitsangiz, u suvga aylanadi.

If I get up early, my father takes me to school.

Agar men erta uyg'onsam, mening otam meni maktabga olib boradi.

Conditional Type 1 (Future)

If + S + Present Simple, S + will + V1.

If I finish work early, I'll get home fast.

Agar ishni erta tugatsam, uyga tez yetib boraman.

If he runs, he'll get there in time.

Agar u yugursa, u yerga vaqtida yetib boradi.

Conditional Type 1 (Modal Verbs)

If + S + Present Simple, S + modal + V1.

If it stops snowing, we can go out.

Agar qor yog'ishi to'xtasa, biz tashqariga chiqishimiz mumkin.

If you want to lose weight, you must/should eat less bread.

Agar sen vazn yo'qotishni xohlasang, kamroq non yeyishing kerak.

Conditional Type 1 (Imperative)

If + S + Present Simple, V1 ... !.

If you see Tom tomorrow, tell him to ring me.

Agar ertaga Tomni ko'rsang, unga qo'ng'iroq qilishimni so'ray.

If you lose the keys, don't open the door.

Agar kalitlarni yo'qotsangiz, eshikni ochmang.

Misollar bilan tanishing

Conditional Type 0 (Faktlar va odatlar)

If you heat water to 100°, it boils.

Agar suvni 100° ga isitsa, u qaynaydi.

If a person is a vegetarian, he/she doesn't eat meat.

Agar biror shaxs vegetarian bo'lsa, u go'sht yemaydi.

The dog is happy if it sees its master.

Agar it o'z xo'jayinini ko'rsa, baxtli bo'ladi.

Conditional Type 1 (Kelajakdagi ehtimollar)

If you try harder, you will succeed in your exams.

Agar qattiqroq harakat qilsang, imtihonlaringda muvaffaqiyatga erishasan.

If you lose your credit card, you will notify the police.

Agar kredit kartangni yo'qotsang, politsiyaga xabar berasan.

If Matthew enters the race, he'll probably win it.

Agar Metyu poygaga qatnashsa, ehtimol u yutadi.

Muhim Qoidalar va Xatolar (Conditionals 0 & 1)

🚫 XATO 1: `if` ergash gapida `will` ishlatish
If it will rain tomorrow, I will stay at home.
If it rains tomorrow, I will stay at home.

Eng asosiy qoida: `if` bilan boshlangan shart qismida hech qachon kelasi zamon (`will`) ishlatilmaydi. Uning o'rniga hozirgi zamon (Present Simple) qo'yiladi.

🚫 XATO 2: Type 0 va Type 1 ni adashtirish
If you heat water, it will boil.
If you heat water, it boils.

Qoida: Ilmiy faktlar, o'zgarmas qonuniyatlar uchun har doim Type 0 (Present + Present) ishlatiladi.

🚫 XATO 3: Vaqt ergash gaplarida `will` ishlatish
I will call you when I will arrive.
I will call you when I arrive.

Qoida: `if` kabi `when`, `as soon as`, `before`, `after`, `until` kabi vaqt bog'lovchilaridan keyin ham kelasi zamon (`will`) o'rniga hozirgi zamon ishlatiladi.

🚫 XATO 4: Noto'g'ri vergul qo'yish
I will go to the party, if I have time.
I will go to the party if I have time.

Qoida: Agar gap `if` ergash gapi bilan boshlansa, undan keyin vergul qo'yiladi. Agar gap bosh gap bilan boshlanib, `if` o'rtada kelsa, vergul qo'yilmaydi.

🚫 XATO 5: Fe'l tuslanishida xato
If he don't study, he will fail.
If he doesn't study, he will fail.

Qoida: `If` qismida Present Simple qoidalariga to'liq rioya qilish kerak. `he/she/it` uchun `doesn't` ishlatiladi.

🚫 XATO 6: `Unless` bilan qo'sh inkor
You won't pass the exam unless you don't study.
You won't pass the exam unless you study.

Qoida: `Unless` so'zining o'zi "if ... not" (agar ... -masa) degan inkor ma'noni beradi. Shuning uchun undan keyingi gapda `don't` yoki `not` ishlatish ortiqcha.

🚫 XATO 7: Type 1 o'rniga Type 0 ishlatish
If I miss the bus, I am late for work.
If I miss the bus, I will be late for work.

Qoida: Bu umumiy odat emas, balki kelajakdagi aniq bir vaziyatga (bugun avtobusni o'tkazib yuborish) bog'liq ehtimoliy natija. Shuning uchun Type 1 ishlatiladi.

🚫 XATO 8: Buyruq maylida `will` ishlatish
If you see him, you will tell him to call me.
If you see him, tell him to call me.

Qoida: Agar bosh gapda buyruq, iltimos yoki maslahat berilayotgan bo'lsa, `will` ishlatilmaydi, to'g'ridan-to'g'ri fe'lning o'zi (buyruq mayli) ishlatiladi.

🚫 XATO 9: `if` va `when` ma'nosini adashtirish
When I pass my exams, I will celebrate. (agar o'tishimga ishonchim komil bo'lmasa)
If I pass my exams, I will celebrate.

Qoida: `if` - agar (ehtimol bor). `when` - qachonki (sodir bo'lishi aniq). Agar imtihondan o'tishingizga hali ishonchingiz komil bo'lmasa, `if` ishlatgan ma'qul.

🚫 XATO 10: Ikkala qismda `will` ishlatish
If you will help me, I will help you.
If you help me, I will help you.

Qoida: Bu eng keng tarqalgan xato. Shuni yodda tuting: `if` ergash gapida kelasi zamon shakli ishlatilmaydi, faqat bosh gapda ishlatiladi.
