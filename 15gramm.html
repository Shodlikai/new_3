Aralash Shart Gaplar Qoidalari

Mixed Conditional nima?

Mixed Conditionals - bu Shart Gaplarning 2- va 3-turlari qismlarini birlashtirib, o'tgan zamondagi shartni hozirgi zamon natijasi bilan yoki hozirgi zamondagi shartni o'tgan zamon natijasi bilan bog'laydigan gaplardir.

1-Turi: O'tgan zamon sharti → Hozirgi zamon natijasi

Ishlatilishi: O'tmishda sodir bo'lmagan (xayoliy) biror harakatning hozirgi kundagi natijasini tasavvur qilish uchun ishlatiladi. "Agar o'tmishda shunday bo'lganida edi, hozir boshqacha bo'lar edi" degan ma'noni beradi.

Formula: If + Past Perfect, ... would + V1

Kalit so'zlar: Natija qismida ko'pincha now, today, at the moment kabi hozirgi zamonga ishora qiluvchi so'zlar keladi.

If you had studied last night, you would know the answer now. (Agar kecha o'qiganingda edi, hozir javobni bilgan bo'larding).

2-Turi: Hozirgi zamon sharti → O'tgan zamon natijasi

Ishlatilishi: Hozirgi kundagi biror (xayoliy) holat yoki shartning o'tgan zamondagi natijasini tasavvur qilish uchun ishlatiladi. "Agar hozir shunday bo'lganimda edi, o'tmishda boshqacha qilgan bo'lar edim" degan ma'noni beradi.

Formula: If + Past Simple, ... would + have + V3

Kalit so'zlar: Natija qismida ko'pincha yesterday, last week, a long time ago kabi o'tgan zamonga ishora qiluvchi so'zlar keladi.

If I weren't afraid of flying, I would have visited you in California a long time ago. (Agar uchishdan qo'rqmaganimda, men anchadan beri Kaliforniyadagi siznikiga tashrif buyurgan bo'lar edim).

Shart Gap Turlari O'rtasidagi Farqlar

1. Real vs. Noreal Shartlar (Type 1 vs. Type 2)

Asosiy farq vaziyatning realligidadir.

  • Type 1 (Real Ehtimol): Kelajakda sodir bo'lishi mumkin.
  • Type 2 (Noreal Xayolot): Hozir sodir bo'lishi deyarli imkonsiz.

Type 1: If I win the lottery, I will buy a new car. (Agar lotereya yutsam, yangi mashina sotib olaman - Bu real reja).

Type 2: If I won the lottery, I would buy a new car. (Agar lotereya yutganimda edi, yangi mashina sotib olardim - Bu shunchaki orzu, yutishimga ishonmayman).

2. Hozirgi Xayolot (Type 2) vs. O'tmishdagi Afsus (Type 3)

Bu ikki noreal shart gapning asosiy farqi ular ishora qilayotgan vaqtdadir.

  • Type 2 (Hozirgi/Kelajak): "Agar hozir shunday bo'lganda edi..." degan ma'noni beradi.
  • Type 3 (O'tmish): "Agar o'tmishda shunday bo'lganida edi..." degan ma'noni beradi.

Type 2: If I spoke French, I would apply for that job. (Agar fransuzcha gapira olganimda edi, o'sha ishga ariza topshirardim - lekin hozir gapira olmayman).

Type 3: If I had spoken French, I would have applied for that job. (Agar fransuzcha gapira olganimda edi, o'sha ishga ariza topshirgan bo'lardim - o'tmishda shunday imkoniyat bo'lgan, lekin til bilmaganim uchun topshirmaganman).

3. Oddiy (Type 3) vs. Aralash (Mixed) Shart Gaplar

Oddiy shart gaplarda zamonlar bir xil bo'ladi (o'tmish sharti-o'tmish natijasi), aralash shart gaplarda esa zamonlar aralashadi.

Type 3: If she had warned me, I wouldn't have been surprised. (Agar u meni ogohlantirganida edi, men hayron qolmagan bo'lardim - o'tmish → o'tmish).

Mixed: If she had warned me, I wouldn't be surprised now. (Agar u meni ogohlantirganida edi, men hozir hayron qolmayotgan bo'lardim - o'tmish → hozir).

Aralash Shart Gap Tuzilmalari

O'tgan Zamon Sharti → Hozirgi Zamon Natijasi

If + S + had + V3, S + would + V1.
If + S + hadn't + V3, S + wouldn't be....

If I had studied last night, I would know the answer now.

Agar siz o'tgan tunda o'qiganingizda, siz hozir savolga javobni bilardik.

If he had left immediately, he would be here now.

Agar u tezda tark etgan bo'lganida, u hozir bu yerda bo'lardi.

Hozirgi Zamon Sharti → O'tgan Zamon Natijasi

If + S + Past Simple, S + would have + V3.
If + S + weren't..., S + would have + V3.

If I weren't afraid of flying, I would have visited you in California a long time ago.

Agar uchishdan qo'rqmaganimda, men ancha vaqt oldin Kaliforniyaga siznikiga tashrif buyurgan bo'lar edim.

If she loved him, she wouldn't have left him.

Agar u uni sevganida, u uni tashlab ketmagan bo'lar edi.

Misollar bilan tanishing

O'tmishdagi Shart → Hozirgi Natija

If she hadn't missed her flight yesterday, she would be here today.

Agar u kecha reysini o'tkazib yubormaganida, u bugun shu yerda bo'lar edi.

If I had studied hard when I was young, I wouldn't be a porter now.

Agar yoshligimda qattiq o'qiganimda, men hozir hammol bo'lmas edim.

If you hadn't eaten my chocolate, I wouldn't be angry.

Agar shokoladimni yegan bo'lganingda, meni jahlim chiqardi.

Hozirgi Shart → O'tmishdagi Natija

If he weren't so lazy, he would have passed the exam.

Agar u bunchalik dangasa bo'lmaganida, imtihondan o'tgan bo'lar edi.

If I wasn't a good cook, I wouldn't have tried to make this cake.

Agar yaxshi oshpaz bo'lganimda, men bu keksni pishirishga urinmagan bo'lar edim.

If Tom had known about the sale, he would shop with us.

Agar Tom chegirmalar haqida bilganida edi, u biz bilan xarid qilardi.

Mashqlar (Mixed Conditionals)

To'g'ri variantni tanlang

1. If I had listened to your advice, I ___ in this mess now.

2. If I were a good student, I ___ for the exam yesterday.

3. If she ___ her flight yesterday, she would be here now.

4. If she spoke French, she ___ that job in Paris last year.

5. If I ___ that job, I would be miserable now.

6. If I weren't so busy, I ___ you with your project yesterday.

7. If I ___ in Spain, I would speak Spanish.

8. If I had more experience, I ___ the project on time last month.

9. If I ___ so much cake last night, I wouldn't feel sick now.

10. If I knew the truth, I ___ you yesterday.

Bo'sh joyni fe'lning to'g'ri shakli bilan to'ldiring.

1. If I had gone to bed earlier last night, I ___ so tired now. (not/be)

2. If she weren't so shy, she ___ to him at the party yesterday. (speak)

3. If you ___ me about the problem, I would be able to help you now. (tell)

4. If I spoke Japanese, I ___ for that job in Tokyo last year. (apply)

5. If they ___ the house, they would be very poor now. (not/sell)

6. If I were you, I ___ that opportunity last week. (not/miss)

7. If he had been born in the US, he ___ a visa to work there. (not/need)

8. If I had more free time, I ___ that book you gave me last month. (read)

9. If I ___ that last piece of pizza, I wouldn't be so hungry now. (eat)

10. If he ___ a better driver, he wouldn't have crashed his car. (be)

Vaziyatga mos shart gap tuzing

1. I don't have money, so I can't buy that car. → (Use: Type 2)

2. I didn't study for the test, so I failed. → (Use: Type 3)

3. I am not you, so I can't give advice. → (Use: Type 2)

5. He doesn't know the answer, so he can't help you. → (Use: Type 2)

Muhim Qoidalar va Xatolar (Conditionals 2, 3 & Mixed)

🚫 XATO 1: `if` ergash gapida `would` ishlatish
If I would have more time, I would travel.
If I had more time, I would travel.

Eng asosiy qoida: `if` bilan boshlangan shart qismida hech qachon `would` yoki `would have` ishlatilmaydi.

🚫 XATO 2: Type 2 va Type 3 ni aralashtirish
If she had studied, she would pass.
If she had studied, she would have passed.

Qoida: Agar `if` qismida Past Perfect (had + V3) bo'lsa va natija ham o'tmishda bo'lsa (Type 3), bosh gapda ham unga mos ravishda `would have + V3` bo'lishi kerak.

🚫 XATO 3: Type 2 da 'was' o'rniga 'were'
If I was you, I would apologize.
If I were you, I would apologize.

Qoida: Type 2 da maslahat berishda yoki xayoliy vaziyatlarda barcha shaxslar uchun `to be` fe'lining `were` shaklini ishlatish grammatik jihatdan to'g'riroq hisoblanadi.

🚫 XATO 4: Type 3 da `have` ni unutish
If I had known, I would gone.
If I had known, I would have gone.

Qoida: Type 3 ning bosh gapida `would` va `V3` o'rtasida `have` yordamchi fe'li bo'lishi shart.

🚫 XATO 5: Type 1 va Type 2 ni adashtirish
If I have a million dollars, I would quit my job.
If I had a million dollars, I would quit my job.

Qoida: Bosh gapda 'would' kelishi bilan bu xayoliy vaziyat (Type 2) ekanligi ma'lum bo'ladi. Shuning uchun 'if' qismida Present Simple emas, Past Simple ishlatilishi kerak.

🚫 XATO 6: Og'zaki nutqdagi xato ('would of')
We would of gone if you had called.
We would have gone if you had called.

Qoida: Og'zaki nutqda `would've` (would have) ning qisqartmasi `would of` kabi eshitilishi mumkin, lekin yozuvda bu jiddiy xato. Har doim `would have` deb yoziladi.

🚫 XATO 7: Type 3 da V3 ni unutish
If he had left earlier, he would have catch the train.
If he had left earlier, he would have caught the train.

Qoida: Type 3 ning bosh gapida 'would have' dan keyin fe'lning 3-shakli (Past Participle - V3) kelishi shart.

🚫 XATO 8: Ikkala qismda `would` ishlatish
If she would have invited me, I would have come.
If she had invited me, I would have come.

Qoida: Bu keng tarqalgan xato. Yodda tuting: 'would' faqat natijani bildiruvchi bosh gapda keladi, 'if' bilan boshlangan shart qismida emas.

🚫 XATO 9: `had had` strukturasidan qo'rqish
If I had a map, I wouldn't have gotten lost.
If I had had a map, I wouldn't have gotten lost.

Qoida: Type 3 da "ega bo'lmoq" (to have) fe'li ishlatilsa, uning Past Perfect shakli `had had` bo'ladi. Birinchi `had` - yordamchi fe'l, ikkinchisi - `have` ning V3 shakli.

🚫 XATO 10: `could/might` bilan `have` ni unutish
If I had studied, I might pass the exam.
If I had studied, I might have passed the exam.

Qoida: Type 3 ning bosh gapida `would` o'rniga `could` yoki `might` ishlatilishi mumkin. Lekin ulardan keyin ham `have + V3` strukturasi saqlanib qolishi shart.

🚫 XATO 11: Aralash shart gapni adashtirish
If I had taken the job, I would have been rich now.
If I had taken the job, I would be rich now.

Qoida: Agar natija hozirgi kunga (`now`) taalluqli bo'lsa, bu Aralash Shart Gap (Mixed Conditional) bo'ladi va bosh gap Type 2 ga o'xshab `would be` bilan yasaladi.

🚫 XATO 12: So'roq shaklida xato
What would you have did if you had seen him?
What would you have done if you had seen him?

Qoida: So'roq gaplarda ham fe'lning 3-shakli (V3) ishlatiladi. `do` fe'lining V3 shakli `done`.

🚫 XATO 13: Inversiya (teskari so'z tartibi)da xato qilish
Had I would have known, I would have come.
Had I known, I would have come.

Qoida: Type 3 da `if` o'rniga inversiya qilish mumkin. Bunda `Had` ega oldiga o'tadi va `if` tushib qoladi. Boshqa o'zgarish bo'lmaydi.

🚫 XATO 14: Type 3 da `would` ni unutish
If I had known, I had come.
If I had known, I would have come.

Qoida: Natijani bildiruvchi bosh gapda har doim 'would' (yoki could/might) modal fe'li bo'lishi kerak. 'had come' faqat 'if' qismida keladi.

🚫 XATO 15: Faqat 'if' so'ziga yopishib olish
I would have helped if I was there.
I would have helped if I had been there.
I would help if I were there.

Qoida: Gapning ikkinchi qismidagi 'would have helped' (Type 3 natijasi) yoki 'would help' (Type 2 natijasi) qaysi tur ekanligini belgilab beradi. 'If' qismini ham o'sha turga moslash kerak.
